# Auto-ML-WebApp

This Auto ML WebApp is a Streamlit-based web application that automates the process of data profiling and machine learning model training using pandas profiling and PyCaret library.

## Features

- Upload your dataset for modeling
- Generate an exploratory data analysis (EDA) report using pandas profiling
- Train machine learning models using PyCaret library
- Save the best model for your classification problem

## Installation

1. Clone the repository:

```bash
git clone https://github.com/joelsiby02/Auto-ML-WebApp.git
```

2. Install the required dependencies. It is recommended to use a virtual environment:

```bash
cd Auto-ML-WebApp
pip install -r requirements.txt
```

## Usage

1. Run the Streamlit app:

```bash
streamlit run app.py
```

2. Access the app in your browser at `http://localhost:8501`.

3. Use the sidebar navigation to perform the following tasks:
   - **Upload**: Upload your dataset for modeling.
   - **Profiling**: Generate an exploratory data analysis report.
   - **ML**: Train machine learning models and save the best model.
   - **Download**: Download the trained model.
